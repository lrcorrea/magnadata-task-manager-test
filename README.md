# TaskManager

Aplicação de gerenciamento de tarefas composta por uma API em ASP.NET Core e um frontend em Nuxt 3.

Este projeto foi desenvolvido como parte de uma avaliação técnica para a MagnaData.

## Estrutura

- TaskManager.Api → Backend (ASP.NET Core)
- TaskManager.Web → Frontend (Nuxt)

## Como rodar

- Backend: veja em TaskManager.Api/README.md
- Frontend: veja em TaskManager.Web/README.md
